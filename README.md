# Quick start
### working example
[https://kubek80.github.io/](https://kubek80.github.io/)

### environment
node v 4.6.0
npm i -g webpack gulp karma-cli

#### clone
`git clone https://github.com/kubek80/NG6-starter.git jz`

#### change directory
`cd jz`

#### install dependencies
`npm i`

#### tests
`npm test`

#### run
`npm start`

#### link (default)
`localhost:3000`
